Welcome to your new markdown blog! 🎉

This page is just a regular **Markdown file** that lives in
`src/pages/posts/hello-world.md`.

- To create a new post, copy this file
- Change the frontmatter at the top
- Write your content in Markdown
- Commit & push to GitHub

Astro will automatically:

1. Turn each `.md` file in `src/pages/posts/` into its own page under `/posts/...`
2. Add it to the list on your homepage.
